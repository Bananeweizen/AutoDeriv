This is the AutoDeriv plugin Eclipse update site. There is no fun content here.

 

Your options:

·         You can check the Home page of AutoDeriv plugin there :

http://nodj.github.io/AutoDeriv/

 

·         You may install this plugin into your IDE by adding this update site.

Go to: Windows > Preference > Install/Update > Available Software Sites,

Click the Add… button, enter the following settings

Site name example        AutoDeriv-update

Update site Location     http://nodj.github.io/AutoDeriv/update

 

Now you can fetch the plugin from the Help > Install new Software, selecting the newly created update site.
